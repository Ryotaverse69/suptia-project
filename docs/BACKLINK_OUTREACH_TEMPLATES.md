# 被リンク獲得 アウトリーチテンプレート集

## 基本方針

- パーソナライズド（相手のサイトを具体的に言及）
- 価値提供を最優先（相互利益を明確に）
- 短く簡潔に（300文字以内）
- 押し付けがましくない

## NGパターン（避けるべき表現）

### ❌ 悪い例

```
件名：被リンクください

初めまして。
サプティアというサイトを運営しています。
被リンクをいただきたく連絡しました。
よろしくお願いします。
```

**問題点**:

- 相手のメリットが不明
- 押し付けがましい
- パーソナライズ皆無
- 短すぎて不誠実

### ✅ 良い例

```
件名：【相互メリット】サプリ比較データの共有について

[サイト名]運営者様

初めまして、サプティアの[名前]と申します。

貴サイトの「[具体的な記事]」を拝見し、
非常に参考になりました。

弊サイトでは[価値提供内容]を
無料で提供しております。

貴サイトの読者様にもお役立ていただけるかと思い、
ご紹介させていただきました。

[具体的な提案]

ご検討いただければ幸いです。
```

## 成功率向上のポイント

### 1. タイミング

- 火曜日〜木曜日の午前中が最適
- 月曜日の朝は避ける
- 週末は避ける

### 2. パーソナライゼーション

- 必ず相手のサイト名・記事タイトルを具体的に言及
- コピペ感を出さない
- 相手の記事を本当に読んだことが伝わる内容に

### 3. 価値提供

- 「お願い」より「提供」を前面に
- 相手のメリットを明確に
- win-winの関係を強調

### 4. 追跡管理

- スプレッドシートで管理
- 送信日、返信日、結果を記録
- 成功率を計測
