# 比較記事 対象キーワード最終決定

**作成日**: 2025-12-17
**期間**: 2026年1月〜3月（各月3本 = 計9本）

---

## 選定基準

1. **検索ボリューム** - 月間500以上を目安
2. **Suptiaの強みを活かせる** - 5軸評価・mg単価比較が差別化になる
3. **商品データの充実度** - 比較対象商品が十分にある（5商品以上）
4. **競合の強さ** - 大手が独占していないキーワード

## 記事制作の優先順位

| 優先度 | キーワード                | 理由                         |
| ------ | ------------------------- | ---------------------------- |
| 1      | ビタミンC サプリ 比較     | 検索ボリューム大、データ豊富 |
| 2      | マルチビタミン コスパ     | 既存ロジック活用可能         |
| 3      | ビタミンD サプリ おすすめ | 季節需要（冬）               |
| 4      | DHC ディアナチュラ 比較   | 指名検索で成約率高い         |
| 5      | 鉄分 サプリ 比較          | 女性需要安定                 |
| 6      | 亜鉛 サプリ おすすめ      | 競合少ない                   |
| 7      | マグネシウム サプリ 比較  | 成長市場                     |
| 8      | 睡眠 サプリ 比較          | 需要高いが競合多い           |
| 9      | 美容 サプリ 5つの評価軸   | 独自切り口の実験             |
