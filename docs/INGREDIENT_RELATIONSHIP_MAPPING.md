## 1. 相乗効果のある組み合わせ

吸収率向上や効果増強が期待できる組み合わせ。内部リンクの優先度高。

| 組み合わせ                    | 相乗効果                   | リンク優先度 |
| ----------------------------- | -------------------------- | ------------ |
| ビタミンD + カルシウム        | カルシウム吸収促進（必須） | S            |
| ビタミンC + 鉄                | 鉄の吸収率3倍向上          | S            |
| ビタミンD + ビタミンK2        | カルシウムを骨に定着       | A            |
| ビタミンE + ビタミンC         | 抗酸化作用の相互再生       | A            |
| ビタミンB12 + 葉酸            | 赤血球生成・神経機能       | A            |
| マグネシウム + ビタミンD      | ビタミンD活性化に必要      | A            |
| 亜鉛 + ビタミンC              | 免疫機能相乗効果           | B            |
| オメガ3 + ビタミンE           | オメガ3の酸化防止          | B            |
| グルコサミン + コンドロイチン | 軟骨修復の相乗効果         | B            |
| カルシウム + マグネシウム     | 骨代謝のバランス維持       | B            |
| ルテイン + ゼアキサンチン     | 黄斑保護の相乗効果         | B            |
| BCAA + ビタミンB6             | アミノ酸代謝促進           | C            |

## 2. 同カテゴリ成分（相互リンク対象）

### 水溶性ビタミン

```
ビタミンC ←→ ビタミンB群（全体）
    ↓
ビタミンB1 ←→ B2 ←→ B3 ←→ B5 ←→ B6 ←→ B7 ←→ B12
                              ↓
                           葉酸
```

**リンク文例**: 「ビタミンB群の他の成分も見る」「水溶性ビタミン一覧」

### 脂溶性ビタミン

```
ビタミンA ←→ ビタミンD ←→ ビタミンE ←→ ビタミンK
```

**リンク文例**: 「脂溶性ビタミンは一緒に摂ると効率的」

### ミネラル

```
カルシウム ←→ マグネシウム ←→ 鉄 ←→ 亜鉛
    ↓              ↓
  リン          カリウム
```

**注意**: 鉄と亜鉛は吸収競合するため、摂取タイミングをずらす推奨も記載

### アミノ酸

```
BCAA（ロイシン + イソロイシン + バリン）
    ↓
グルタミン ←→ アルギニン ←→ シトルリン
```

## 4. 内部リンク実装ルール

### 自動リンク化の優先順位

1. **相乗効果ペア（Sランク）** - 必ずリンク + 説明文追加
2. **同カテゴリ成分** - 「関連成分」セクションでリンク
3. **目的別グループ** - 「〇〇が気になる方へ」でリンク

### リンク文テンプレート

```tsx
// 相乗効果ペアの場合
<p>
  ビタミンDは<Link href="/ingredients/calcium">カルシウム</Link>
  の吸収を促進するため、一緒に摂取することをおすすめします。
</p>

// 同カテゴリの場合
<section>
  <h3>関連する水溶性ビタミン</h3>
  <ul>
    <li><Link href="/ingredients/vitamin-b1">ビタミンB1</Link></li>
    ...
  </ul>
</section>

// 目的別の場合
<section>
  <h3>疲労回復が気になる方におすすめの成分</h3>
  ...
</section>
```

## 5. 注意すべき組み合わせ（吸収阻害）

内部リンク時に注意喚起を追加する組み合わせ。

| 組み合わせ                | 阻害効果       | 対策                      |
| ------------------------- | -------------- | ------------------------- |
| 鉄 + カルシウム           | 吸収競合       | 摂取時間を2時間以上空ける |
| 鉄 + 亜鉛                 | 吸収競合       | 同時摂取を避ける          |
| カルシウム + マグネシウム | 高用量で競合   | 2:1の比率を推奨           |
| 鉄 + お茶・コーヒー       | タンニンが阻害 | 食事と一緒に摂取          |
